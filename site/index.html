CortenForge

A software development kit for the mechatronics and simulation space.

Composable Rust components for the full path from physical to digital and back — geometry, design, meshing and fabrication, rigid- and soft-body physics, control, and reinforcement learning.

Featured app · built with CortenForge

Cendrillon

A guided desktop tool that takes a 3D scan all the way to printable, silicone-cast molds, step by step.

Download for MacApple Silicon · tested Download for Windowsbeta Download for Linuxbeta

On an older Intel Mac? The Intel build is in beta — find it (and every other download) on the releases page.

First time you open it (it's unsigned for now)
  • Mac — unzip, drag Cendrillon.app into Applications, and double-click. macOS will say it can’t verify the developer: open System Settings → Privacy & Security, scroll down, and click Open Anyway, then open it again. You only do this once.
  • Windows — unzip and run cf-studio-gui.exe. If SmartScreen warns, click More info → Run anyway.
  • Linux — tar xzf and run the cf-studio-gui binary (needs a working Vulkan/GL driver).

Code-signing & notarization (a one-click launch) is a planned upgrade.

Prefer to check first? The full source is on GitHub — you can read every line before you download.
