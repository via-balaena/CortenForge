# §41 Golden-File Flag Conformance Data

Golden .npy files generated by MuJoCo for verifying CortenForge's runtime flag
behavior matches the reference implementation.

## Regenerating

```bash
uv pip install mujoco numpy
uv run sim/L0/tests/scripts/gen_flag_golden.py
```

## Files

- `flag_golden_test.xml` — canonical MJCF model (1-DOF hinge + spring/damper/motor/floor)
- `baseline_qacc.npy` — qacc per step, no flags set (10 steps, shape `(10, nv)`)
- `disable_gravity_qacc.npy` — qacc per step with DISABLE_GRAVITY (10 steps)

## Format

NumPy v1.0 `.npy` — little-endian float64, C-contiguous, shape `(nsteps, nv)`.
